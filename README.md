# Tax Calculator AI MCP

Estimate income tax liability across multiple jurisdictions with support for brackets, deductions, and effective rate calculations.

## Tools

- `calculate_us_federal_tax` — Calculate approximate US federal income tax for a given filing status and income.
- `calculate_uk_income_tax` — Calculate approximate UK income tax for a given annual salary.
- `calculate_effective_tax_rate` — Compute the effective tax rate given total tax and gross income.

## Install

```bash
pip install -e .
```

## Run

```bash
python -m tax-calculator-ai-mcp.server
```
